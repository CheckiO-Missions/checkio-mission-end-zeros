Je n'ai aucune idée de la manière de résoudre cette mission

Vous devez comparer deux chaînes de caractères. L'une est le même nombre que la chaîne, et l'autre est le même nombre que la chaîne, mais sans les zéros ("0") à la fin.

Quelle fonction dois-je rechercher?

Ces fonctions vous suffiront pour résoudre cette mission.

str() vous aide à convertir n'importe quel objet en chaîne de caractères.

str.strip() méthode de conversion d'un objet de type chaîne de caractères. Elle renvoie une copie de la chaîne de caractères dont les caractères de début et de fin ont été supprimés.

len() est la fonction qui permet de trouver la longueur

D'accord. Comment puis-je l'utiliser alors? 

number = 5600
text = str(number)
text_length = len(text)
# la variable text_length contient maintenant la longueur de la chaîne de caractères texte

text_without_zeros = text.strip('0')
text_length_without_zeros = len(text_without_zeros)

Une dernière question. A quoi ressemblera ma solution finale? 

def end_zeros(num: int) -> int:

    text = str(num)
    text_length = len(text)
    # la variable text_length contient maintenant la longueur de la chaîne de caractères texte

    text_without_zeros = text.strip('0')
    text_length_without_zeros = len(text_without_zeros)
    
    return text_length - text_length_without_zeros

Encore plus simple

def end_zeros(num: int) -> int:

    n = str(num)

    return len(n) - len(n.strip('0'))
