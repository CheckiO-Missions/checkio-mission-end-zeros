Nie mam pojęcia, jak zacząć rozwiązywać tę misję

Proszę porównać dwa ciągi. Jeden z nich to liczba w postaci ciągu, a drugi to ta sama liczba w postaci ciągu, ale bez zer ("0") na końcu.

Której funkcji mam szukać?

Te funkcje wystarczą Państwu do rozwiązania tego zadania.

str() pomaga przekonwertować dowolny obiekt na ciąg znaków.

str.strip() metoda obiektu string. Zwraca kopię ciągu znaków z usuniętymi znakami początkowymi i końcowymi.

len() jest funkcją do znajdowania długości

Ok. Jak w takim razie mogę jej użyć? 

text = str(number)
text_length = len(text)
# the variable text_length now contains length of the string text

text_without_zeros = text.strip('0')
text_length_without_zeros = len(text_without_zeros)

Ostatnie pytanie. Jak będzie wyglądało moje ostateczne rozwiązanie?

Teraz musi Pan obliczyć różnicę między długością tekstu z zerami a długością tekstu bez zer. Proszę więc zawinąć swój kod w wyrażenie funkcyjne i zwrócić obliczoną różnicę, która jest wynikiem. Zdecydowanie zaleca się używanie podpowiedzi typu dla argumentu wejściowego i wartości wyjściowej.

def end_zeros(num: int) -> int:

    text = str(num)
    text_length = len(text)
    # the variable text_length now contains length of the string text

    text_without_zeros = text.strip('0')
    text_length_without_zeros = len(text_without_zeros)
    
    return text_length - text_length_without_zeros

Jeszcze prostsza wersja bez deklarowania niepotrzebnych zmiennych:

def end_zeros(num: int) -> int:

    n = str(num)

    return len(n) - len(n.strip('0'))
