您需要比较两个字符串。一个是与字符串相同的数字，另一个是与字符串相同的数字，但末尾没有零（"0"）。

str() 帮助您将任何对象转换为字符串。

str.strip() 字符串对象的方法。返回去掉前导字符和尾部字符的字符串副本。

len() 是查找字符串长度的函数

最后一个问题。我的最终解决方案是什么样的？

现在，您需要计算带 0 的文本长度与不带 0 的文本长度之间的差值。因此，将代码封装到函数表达式中，然后返回计算出的差值，这就是结果。强烈建议对输入参数和输出值使用类型提示。

def end_zeros(num: int) -> int:

    text = str(num)
    text_length = len(text)
    # the variable text_length now contains length of the string text

    text_without_zeros = text.strip('0')
    text_length_without_zeros = len(text_without_zeros)
    
    return text_length - text_length_without_zeros

更简单的版本无需声明不必要的变量：

def end_zeros(num: int) -> int:

    n = str(num)

    return len(n) - len(n.strip('0'))
