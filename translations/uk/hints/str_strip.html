Я не маю жодного уявлення, як почати вирішувати цю місію

Вам потрібно порівняти два рядки. Один з них - число у вигляді рядка, а інший - таке ж число у вигляді рядка, але без нулів ("0") у кінці.

Яку функцію я шукаю?

Вам буде достатньо цих функцій, щоб розв'язати цю задачу.

str() допомагає конвертувати будь-який об'єкт у рядок.

str.strip() метод рядкового об'єкту. Повертає копію рядка з видаленими початковими та кінцевими символами.

len() функція для знаходження довжини

Гаразд. Як її використовувати? 

text = str(number)
text_length = len(text)
# the variable text_length now contains length of the string text

text_without_zeros = text.strip('0')
text_length_without_zeros = len(text_without_zeros)

І останнє питання. Як буде виглядати моє остаточне рішення?

Тепер вам потрібно обчислити різницю між довжиною тексту з нулями і довжиною тексту без нулів. Тому оберніть ваш код у вираз функції і поверніть цю обчислену різницю, яка і буде вашим результатом. Наполегливо рекомендується використовувати підказки типів для вхідного аргументу та вихідного значення.

def end_zeros(num: int) -> int:

    text = str(num)
    text_length = len(text)
    # the variable text_length now contains length of the string text

    text_without_zeros = text.strip('0')
    text_length_without_zeros = len(text_without_zeros)
    
    return text_length - text_length_without_zeros

Ще простіший варіант без оголошення зайвих змінних:

def end_zeros(num: int) -> int:

    n = str(num)

    return len(n) - len(n.strip('0'))
