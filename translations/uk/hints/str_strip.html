Я не можу зрозуміти з чого мені розпочати...
Тобі потрібно порівняти дві строки. Одна являє собою число у вигляді строки, а інша являє собою те саме число у вигляді строки, але вже без нулів ("0") у кінці.
Яку саме функцію мені потрібно використовувати?

Цих функцій буде більш ніж достатньо для тебе, що вирішити цю місію.

str допоже тобі конвертувати будь який об'єкт у строку.

str.strip метод строкового об'єкту. Повертає копію строки з видаленими початковими та кінцевими символами.

len це функція для пошуку довжини об'єкту (якщо об'єкт підтримує вимір довжини).

Добре. Як мені використати ці функції для рішення місії? 

number = 5600
text = str(number)
text_length = len(text)
# the variable text_length now contains length of the string text

text_without_zeros = text.strip('0')
text_length_without_zeros = len(text_without_zeros)
Останнє запитання. Як має виглядати мій фінальний код? 
def end_zeros(num: int) -> int:
    text = str(num)
    text_length = len(text)
    # the variable text_length now contains length of the string text

    text_without_zeros = text.strip('0')
    text_length_without_zeros = len(text_without_zeros)
    
    return text_length - text_length_without_zeros

Even simplier

def end_zeros(num: int) -> int:
    n = str(num)
    return len(n) - len(n.strip('0'))
