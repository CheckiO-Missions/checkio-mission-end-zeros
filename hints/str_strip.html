Now you need to calculate the difference between the length of the text with zeros and the length of the text without zeros. So wrap your code into fuction expression and return this calculated difference which is your result. It is strongly recommended to use type hints for input argument and output value.

def end_zeros(num: int) -> int:

    text = str(num)
    text_length = len(text)
    # the variable text_length now contains length of the string text

    text_without_zeros = text.strip('0')
    text_length_without_zeros = len(text_without_zeros)
    
    return text_length - text_length_without_zeros

Even simpler version without declaring unnecesary variables:

def end_zeros(num: int) -> int:

    n = str(num)

    return len(n) - len(n.strip('0'))
