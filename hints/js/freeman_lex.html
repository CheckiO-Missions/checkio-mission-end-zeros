First, you should declare variable num_zeros with initial value 0 to count end zeros. Second, since you need to repeat some operations, until some condition is true, while loop is optimal in this case. 

let num_zeros = 0;
while (...) {
    ...

How to write correct condition for loop?

You have to perform operations, if the reminder from the division of number by 10 is equal 0. If it's not, it means that no end zeros left.

Find the reminder with modulus operator. 

(a%10)

If reminder is 0 (what you need), as a condition it's considered as false (loop is not work) and vise versa. To make loop works, you need to invert this connection - 0 reminder should give true condition. Do it with logical operator ! (not). 

(!(a%10))

I don't understand what to do inside loop.

You get inside loop if the condition is true - the reminder from division (the last digit of number) equals 0. That's what you need to count, so increase num_zeros with increment operator. 

let num_zeros = 0;
while (!(a%10)) {
    num_zeros++;
    ...
}
My loop looks infinite in case when number ends with zero. How to make it works correctly?

When the number ends with 0, you need to "trim" this zero from the number. As i said in the first hint, you may do it by re-assigning you number variable to the quotient from the division number by 10. Since when you divide any number by 10, you visually move comma to the left, "trimming" the last digit (it's the reminder now). To get such quotient, round down the result of division number by 10 with Math.floor() method. 

let num_zeros = 0;
while (!(a%10)) {
    a = Math.floor(a/10);
    num_zeros++;
}

One more step to avoid infinite loop is to take case of case when number equals 0 initially. In this case condition for loop is true initially and quotient always remains equal 0, making condition always true.

Since, number 0 still has one "end" zero, in this case you should return 1, and do this checking before loop. 

if (a==0)
    return 1;

So, how my final solution should look like?

You just need to wrap your code into function expression and return num_zeros.

Here is the whole solution:

function endZeros(a: number): number {

    if (a==0) // or (!a)
        
        return 1;
    
    let num_zeros = 0;
    while (!(a%10)) {
        a = Math.floor(a/10);
        num_zeros++;
    }
    return num_zeros;
}

Convert number into string, replace zeros from end with nothing and find difference of lengths of two string. 

function endZeros(a: number): number {

    const s = String(a);

    return s.length - s.replace(/0*$/, '').length;
}
